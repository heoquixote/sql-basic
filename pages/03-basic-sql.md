SQL의 가장 기본이 되는 조회 문법을 다루는 챕터입니다.

## 이 챕터의 목표

- SQL 쿼리의 기본 흐름을 설명할 수 있습니다.
- 데이터를 조회, 필터링, 정렬, 그룹화하는 기본 문법을 사용할 수 있습니다.
- 각 절이 어떤 역할을 하는지 문장 단위가 아니라 실행 흐름 기준으로 이해할 수 있습니다.

## 세부 목차

- 03-1. SQL 쿼리 작성 순서
- 03-2. 데이터 조회 (SELECT ~ FROM)
- 03-3. 데이터 필터링 (WHERE)
- 03-4. 데이터 정렬, 범위 제한 (ORDER BY, LIMIT)
- 03-5. 데이터 그룹화, 그룹 조건 (GROUP BY, HAVING)
