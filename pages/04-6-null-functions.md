## 2. 문제

### 1. 본인의 사원번호와 매니저의 사원번호를 아래와 같이 출력하는 SQL 쿼리를 작성하시오. (매니저 사원번호가 없다면 본인 사원번호를 출력)

| 사원번호 | 매니저사번 |
| --- | --- |
| 1000 | 1000 |
| 4001 | 1000 |
| 4002 | 4001 |
| 7001 | 1000 |
| 7002 | 7001 |
| 7003 | 7001 |
| 7004 | 7003 |
| 7005 | 7002 |
| 7006 | 7002 |
| 7007 | 7002 |
| 7008 | 7003 |
| 7009 | 7002 |
| 7010 | 7003 |
| 8001 | 8001 |

<details>
<summary>정답 보기</summary>

```sql
SELECT empno AS '사원번호'
     , IFNULL(mgr, empno) AS '매니저사번'
FROM emp;
```
