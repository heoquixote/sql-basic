## 2. 문제

### 1. 급여가 6000 이상인 사원 정보를 아래와 같이 급여(연봉)와 월급으로 출력하는 SQL문을 작성하시오.

*월급은 연봉 / 12로 가정합니다.*

(월급 기준으로 내림차순 정렬하고, 월급은 소수점 0자리 수까지 **버림**하여 표시하시오)

| 이름[직급] | 연봉 | 월급 |
| --- | --- | --- |
| 김대표 [CEO] | 80000 | 6666 |
| 백정태 [상무] | 20000 | 1666 |
| 김낙수 [부장] | 15000 | 1250 |
| 최재혁 [부장] | 13000 | 1083 |
| 도진우 [부장] | 12000 | 1000 |
| 신준섭 [차장] | 10000 | 833 |
| 허태환 [과장] | 8000 | 666 |
| 송익현 [과장] | 7000 | 583 |
| 정성구 [대리] | 6000 | 500 |
| 이주영 [작업반장] | 6000 | 500 |

- 정답

```sql
SELECT CONCAT(ename_ko, ' [', joblv, ']') AS '이름[직급]'
     , sal AS '연봉'
     , TRUNCATE(sal / 12, 0) AS '월급'
FROM emp
WHERE sal >= 6000
ORDER BY 3 DESC;
```

### 2. 사원의 성과금을 연봉으로 나누어 보너스 비율을 구하려고 합니다.

보너스 비율은 아래와 같이 소수점 2번째 자리까지 반올림하여 표시해주세요.

(성과금 0원이면 제외, 보너스 비율 내림차순 정렬)

| 이름 | 연봉 | 성과금 | 보너스 비율 |
| --- | --- | --- | --- |
| 도진우 | 12000 | 7000 | 58.33% |
| 백정태 | 20000 | 10000 | 50.00% |
| 최재혁 | 13000 | 5000 | 38.46% |
| 김낙수 | 15000 | 5000 | 33.33% |
| 이주영 | 6000 | 2000 | 33.33% |
| 신준섭 | 10000 | 3000 | 30.00% |
| 허태환 | 8000 | 2000 | 25.00% |
| 채소연 | 4500 | 1000 | 22.22% |
| 송익현 | 7000 | 1500 | 21.43% |
| 권송희 | 4800 | 1000 | 20.83% |

- 정답

```sql
SELECT ename_ko AS '이름'
     , sal AS '연봉'
     , comm AS '성과금'
     , CONCAT(ROUND(comm / sal * 100, 2), '%') AS '보너스 비율'
FROM emp
WHERE comm != 0
ORDER BY 4 DESC;
```
