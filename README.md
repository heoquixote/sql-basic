# Learners School : SQL 기초

SQL을 처음 배우는 학습자가 데이터베이스 구조부터 조회, 가공, 집계, 결합, 서브쿼리와 CTE까지 단계적으로 익힐 수 있도록 구성한 실습형 입문서 프로젝트입니다.  
문법만 나열하는 방식이 아니라, 비즈니스 질문을 데이터 구조로 해석하고 SQL로 표현하는 흐름까지 함께 다룹니다.

## 이 책이 다루는 것

- 데이터베이스, 테이블, 컬럼, 행을 이해하는 기본 구조
- MySQL과 DBeaver를 활용한 실습 환경 구성
- `SELECT`, `WHERE`, `GROUP BY`, `HAVING`, `ORDER BY`, `LIMIT` 같은 기초 SQL
- 문자열, 숫자형, 변환형 함수와 조건문
- 윈도우 함수, JOIN, UNION
- 중첩 서브쿼리, 스칼라 서브쿼리, 인라인 뷰, CTE

## 이런 분께 적합합니다

- SQL을 처음 배우는 분
- 데이터 분석, BI, 리포트 작성 업무를 시작하는 분
- Tableau, Excel, 대시보드 도구를 쓰다가 SQL이 필요해진 분
- 실무형 예제로 SQL 기본기를 빠르게 잡고 싶은 분

## 학습 목표

이 책을 마치면 다음 내용을 직접 작성할 수 있습니다.

- 테이블 구조를 보고 필요한 데이터를 어디서 가져와야 하는지 판단하기
- 조건 필터링, 정렬, 집계, 그룹 조건을 포함한 기본 조회 쿼리 작성하기
- 함수와 조건문으로 결과를 가공하고 읽기 좋게 표현하기
- 윈도우 함수로 순위, 누적값, 이전/다음 행 비교하기
- JOIN, UNION, 서브쿼리, CTE로 복잡한 쿼리를 단계적으로 분해하기

## 도서 구성

### 1. 데이터베이스와 테이블 이해

데이터베이스, 스키마, 테이블, 컬럼, 행 같은 가장 기본적인 구조를 정리합니다.  
SQL 문법을 배우기 전에 “무엇을 조회하는가”를 구조적으로 이해하는 장입니다.

### 2. DBeaver 설치 및 실습 환경 구성

MySQL 설치, DBeaver 연결, 실습 데이터 적재 과정을 다룹니다.  
처음 학습하는 분도 바로 쿼리를 실행할 수 있도록 실습 환경 중심으로 구성합니다.

### 3. 기초 SQL

가장 핵심이 되는 조회 문법을 단계적으로 학습합니다.  
쿼리 작성 순서, `SELECT ~ FROM`, `WHERE`, `ORDER BY`, `LIMIT`, `GROUP BY`, `HAVING`을 다룹니다.

### 4. 내장 함수

문자열, 숫자형, 날짜형, 변환형, Null 관련 함수를 통해 데이터를 가공하는 방법을 익힙니다.  
단순 문법 암기보다 “언제 어떤 함수를 써야 하는가”에 초점을 둡니다.

### 5. 조건문

`IF`, `CASE`를 중심으로 SQL 안에서 조건에 따라 값을 분기하는 방법을 정리합니다.  
분류, 라벨링, 구간화 같은 실무 예제로 연결되는 장입니다.

### 6. 윈도우 함수

집계는 하되 행을 잃지 않는 SQL의 핵심 분석 기능을 다룹니다.  
순위, 누적 평균, 이전 값, 다음 값 조회를 실습합니다.

### 7. 조인과 유니온

테이블과 결과 집합을 결합하는 두 가지 핵심 방법을 설명합니다.  
특히 JOIN은 열을 옆으로 붙이고, UNION은 행을 아래로 붙인다는 관점을 실습 중심으로 이해합니다.

### 8. Sub Query & CTE

복잡한 SQL을 더 읽기 쉽고 단계적으로 작성하는 방법을 다룹니다.  
중첩 서브쿼리, 스칼라 서브쿼리, 인라인 뷰, CTE를 비교하며 학습합니다.

상세 목차는 [TOC.md](./TOC.md)에서 확인할 수 있습니다.
